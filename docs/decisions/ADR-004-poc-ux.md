# ADR-004 — Linguagem visual da POC

## Status

Aceita.

## Decisão

Usar uma experiência simples, amigável e lúdica inspirada em aplicativos educacionais modernos, com cartões arredondados, progresso visual e uma ação principal por tela.

## Limite

Não copiar elementos distintivos do Duolingo, incluindo mascote, ilustrações, paleta exata, tipografia, sons ou composição de telas.
